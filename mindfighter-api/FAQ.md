# FAQ - Frequently Asked Questions

## General

**Q: What is Mindfighter API?**
A: Mindfighter API is a Python-based FastAPI application for managing digital product tasks.

**Q: What are the system requirements?**
A: Python 3.8+, pip, and a virtual environment (recommended).

**Q: How do I get started?**
A: Follow the SETUP.md guide for installation instructions.

## Installation & Setup

**Q: I'm getting a "ModuleNotFoundError". What should I do?**
A: Run `pip install -r requirements.txt` to install dependencies.

**Q: Can I use Python 3.7?**
A: We recommend Python 3.8 or higher.

## Development

**Q: How do I run tests?**
A: Use `pytest` or `python -m unittest discover`.

**Q: How do I contribute?**
A: See CONTRIBUTING.md for guidelines.

## Deployment

**Q: Can I deploy to Heroku?**
A: Yes! See DEPLOYMENT.md for instructions.

**Q: Does it support Docker?**
A: Yes, a Dockerfile is provided.

## Support

**Q: Where can I get help?**
A: Check the documentation or open an issue on GitHub.
